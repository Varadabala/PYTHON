ii)Lists:
üìå What is a List?

A list is an ordered, mutable collection of elements.

‚úî Can store different data types
‚úî Allows duplicate values
‚úî Uses square brackets [ ]

COMPLETE LIST OF LIST METHODS

Python list has 11 built-in methods.

1Ô∏è‚É£ append(x)

Adds one element at the end.

l.append(3)
print(l)


Output:

[1, 2, 3]

2Ô∏è‚É£ extend(iterable)

Adds multiple elements to the list.

l = [1, 2]

print(l)


Output:

[1, 2, 3, 4]

3Ô∏è‚É£ insert(index, x)

Inserts element at a specific position.

print(l)


Output:

[1, 2, 3]

4Ô∏è‚É£ remove(x)

Removes first occurrence of value.

l.remove(2)
print(l)


‚ùå Error if element not found.

5Ô∏è‚É£ pop([index])

Removes element at index
(Default: last element)

print(l)


Output:

[10, 20]

6Ô∏è‚É£ clear()

Removes all elements.

print(l)


Output:

7Ô∏è‚É£ index(x)

Returns index of element.

1


‚ùå Error if not found.

8Ô∏è‚É£ count(x)

Counts occurrences of element.

Output:

9Ô∏è‚É£ sort()

Sorts list in ascending order.

Output:

üîü reverse()

Reverses list order.

1Ô∏è‚É£1Ô∏è‚É£ copy()

iii)TUPLE

üìå What is a Tuple?

A tuple is an ordered, immutable collection of elements.

‚úî Uses parentheses ( )
‚úî Allows duplicate values
‚úî Faster than list
‚ùå Cannot be modified (no add/remove)

Example
t = (10, 20, 30, 20)

üîπ TUPLE METHODS (ONLY 2)
1Ô∏è‚É£ count(x)

Returns number of occurrences of an element.

Program
t = (10, 20, 30, 20, 20)

Output:3

2Ô∏è‚É£ index(x)

Returns index of first occurrence of an element.

Program

Output:2

‚ùå Error if element not found.


3)Non-Sequential Elements

SET (set) DATA TYPE IN PYTHON
üìå What is a Set?

A set is an unordered collection of unique elements.

‚úî No duplicate values
‚úî Mutable
‚úî Uses { }
‚ùå No indexing / slicing

Example
s = {1, 2, 3, 4}

üîπ ALL SET METHODS (IMPORTANT)
1Ô∏è‚É£ add(x)

Adds one element to the set.

s = {1, 2}
s.add(3)
print(s)

2Ô∏è‚É£ update(iterable)

Adds multiple elements.

s = {1, 2}
s.update([3, 4])
print(s)

3Ô∏è‚É£ remove(x)

Removes element.
‚ùå Error if not present.

s = {1, 2, 3}
s.remove(2)
print(s)

4Ô∏è‚É£ discard(x)

Removes element.
‚úî No error if not present.

s = {1, 2, 3}
s.discard(5)
print(s)

5Ô∏è‚É£ pop()

Removes random element.

s = {10, 20, 30}
s.pop()
print(s)

6Ô∏è‚É£ clear()

Removes all elements.

s = {1, 2, 3}
s.clear()
print(s)

7Ô∏è‚É£ copy()

Returns a copy of the set.

s1 = {1, 2}
s2 = s1.copy()
print(s2)

üîπ SET MATHEMATICAL METHODS
8Ô∏è‚É£ union()

Returns union of sets.

a = {1, 2}
b = {2, 3}
print(a.union(b))

9Ô∏è‚É£ intersection()

Common elements.

print(a.intersection(b))

üîü difference()

Elements in first set only.

print(a.difference(b))

1Ô∏è‚É£1Ô∏è‚É£ symmetric_difference()

Elements in either, but not both.

print(a.symmetric_difference(b))

üîπ RELATIONSHIP METHODS
1Ô∏è‚É£2Ô∏è‚É£ issubset()

Checks subset.

a = {1, 2}
b = {1, 2, 3}
print(a.issubset(b))

1Ô∏è‚É£3Ô∏è‚É£ issuperset()

Checks superset.

print(b.issuperset(a))

1Ô∏è‚É£4Ô∏è‚É£ isdisjoint()

Checks if no common elements.

x = {1, 2}
y = {3, 4}
print(x.isdisjoint(y))



DICTIONARY (dict) DATA TYPE IN PYTHON
üìå What is a Dictionary?

A dictionary stores data in key : value pairs.

‚úî Keys are unique
‚úî Values can be duplicated
‚úî Mutable
‚úî Uses { }

Example
d = {"roll": 101, "name": "Ravi", "marks": 85}

üîπ ALL DICTIONARY METHODS (COMPLETE LIST)

Python dictionary has 11 important methods.

1Ô∏è‚É£ keys()

Returns all keys.

d = {"a": 10, "b": 20}
print(d.keys())


Output:

dict_keys(['a', 'b'])

2Ô∏è‚É£ values()

Returns all values.

print(d.values())


Output:

dict_values([10, 20])

3Ô∏è‚É£ items()

Returns key‚Äìvalue pairs.

print(d.items())


Output:

dict_items([('a', 10), ('b', 20)])

4Ô∏è‚É£ get(key)

Returns value of key.
‚úî No error if key not found.

d = {"a": 10}
print(d.get("a"))
print(d.get("b"))


Output:

10
None

5Ô∏è‚É£ update()

Adds or updates key‚Äìvalue pairs.

d = {"a": 10}
d.update({"b": 20})
print(d)


Output:

{'a': 10, 'b': 20}

6Ô∏è‚É£ pop(key)

Removes key and returns its value.

d = {"a": 10, "b": 20}
d.pop("a")
print(d)


Output:

{'b': 20}


‚ùå Error if key not found.

7Ô∏è‚É£ popitem()

Removes last inserted key‚Äìvalue pair.

d = {"a": 10, "b": 20}
d.popitem()
print(d)

8Ô∏è‚É£ clear()

Removes all items.

d = {"a": 10}
d.clear()
print(d)


Output:

{}

9Ô∏è‚É£ copy()

Returns a shallow copy.

d1 = {"a": 10}
d2 = d1.copy()
print(d2)

üîü setdefault(key, value)

Returns value if key exists;
else inserts key with given value.

d = {"a": 10}
d.setdefault("b", 20)
print(d)


Output:

{'a': 10, 'b': 20}

1Ô∏è‚É£1Ô∏è‚É£ fromkeys(keys, value)

Creates dictionary from keys.

keys = ["a", "b", "c"]
d = dict.fromkeys(keys, 0)
print(d)


Output:

{'a': 0, 'b': 0, 'c': 0}
